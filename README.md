# OpenStack noris nSC Ansible Examples
This repository implements the [noris Sovereign Cloud (nSC) - OpenStack & S3 Guide](https://servicenow.noris.net/csm?id=kb_article_view&sysparm_article=KB0011715) using Ansible with [openstack.cloud collection](https://docs.ansible.com/projects/ansible/latest/collections/openstack/cloud/index.html). Every CLI command from the nSC documentation is converted to idempotent playbooks.

## Usage

Please make sure to place the clouds.yaml in `~/.config/openstack/clouds.yaml`
